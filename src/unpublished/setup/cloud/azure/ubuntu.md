## Setup your Azure configuration

### Install Azure client tool

Open a terminal with <Ctrl>+<Alt>+t and run the following commands:
```sh
$ sudo apt update
$ sudo apt install nodejs npm nodejs-legacy
$ sudo npm install -g azure-cli
```
